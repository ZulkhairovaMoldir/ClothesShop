# Online Clothing Store

Full-stack web application for an online clothing store with basic e-commerce functionality including authentication, product management, and order processing.

---

## Overview

This project represents a simple e-commerce platform where users can browse products, add items to a cart, and complete purchases.
The application includes both frontend and backend components with a relational database.

---

## Features

* User registration and authentication
* Product catalog browsing
* Shopping cart functionality
* Order creation and checkout
* CRUD operations for products and users
* Database-driven application logic

---

## Tech Stack

* Go (Golang)
* SQL (relational database)
* HTML, CSS
* Server-side rendering / basic web templates

---

## Architecture

The application follows a simple full-stack structure:

* Backend handles business logic and database interaction
* Frontend renders UI and interacts with backend endpoints
* Database stores users, products, and orders
